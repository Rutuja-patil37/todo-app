# Todo App (MERN)

A simple Todo application built using:
- Node.js
- Express.js
- MongoDB
- HTML, CSS, JavaScript

## Features
- Add Todo
- Update Todo
- Delete Todo
- View Todos

## How to Run Locally
1. Install Node.js
2. Run `npm install`
3. Create `.env` file with MongoDB URI
4. Run `node server.js`

## Note
MongoDB connection is required to fully run backend.
